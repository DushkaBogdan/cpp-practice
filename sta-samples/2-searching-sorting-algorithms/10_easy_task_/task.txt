1.	Дополните реализацию АТД “множество” для целых чисел на основе односвязных списков следующими операциями:
		
       	IS_SUBSET ( set1, set2 ) : bool  - возвращает true, если первое множество является подмножеством второго множества.

       	EQUAL ( set1, set2 ) : bool  - возвращает true, если два множества состоят из одних и тех же элементов.

	MIN_KEY ( set ) : value  - возвращает ключ с минимальным значением.

	MAX_KEY ( set ) : value  - возвращает ключ с максимальным значением.

2.	В программе имеется заданное в явном виде множество из 20 фамилий, упорядоченных по алфавиту. 
	Фамилии могут повторяться. Пользователь вводит строку, означающую фамилию. Программа ищет введенную 
	фамилию во множестве и печатает количество записей с такой фамилией. Если записей не обнаружено, 
	программа печатает число 0. Обязательное условие - программа в худшем случае должна сделать не более 
	5 сравнений фамилий!
